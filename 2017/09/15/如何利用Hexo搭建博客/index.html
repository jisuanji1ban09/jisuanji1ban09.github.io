如何利用Hexo+github搭建个人博客

##注：本文章仅针对Windows系统用户


  由于工作的需要在github上搭建一个属于个人的博客，就在同事的帮助下及网上各种信息的查找中搭建了一个个人博客，为了其他人不在这个过程中踩坑，所以将具体步骤写出来方便想要哪些搭建个人博客的，同时也是本博客的第一篇文章。


## 必备工具 ##
1. Node.js 官网
2. Git.exe 下载
3. github账号



## Node.js ##

  Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。

  Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。

  Node.js 的包管理器 npm，是全球最大的开源库生态系统。

  Node.js的下载及安装：打开官网，在首页中选用左侧的Recommended For Most Users。
为了可以傻瓜式的Copy,在此特意将本人的下载版本链接放在这里(下载),
Node.js的安装可以选择傻瓜式的一路Next，直至Finish。在Node.js的安装过程中会自动配置好环境变量，所以在这里大家就不用再次去设置环境变量了。

## Git.exe ##

  Git.exe是用于github的commit和clone的作用。

  Git.exe的下载及安装：下载，进入该页面后会自动弹框通知您下载与您操作系统位数相同的安装包。本人下载版本(下载)。Git.exe您一样可以选择傻瓜式安装直至结束。


## github账号 ##

  如果您现在还没有github的账号，那么您需要到github注册个新的账号。我想这个操作是肯定没有问题的，那么我们就继续往下走吧。


## Hexo的安装 ##

  在适合的位置创建个文件夹用于安装Hexo(强烈建议使用英文而非中文)，例如：D:\hexo。在该文件夹下Shif+鼠标右键打开cmd或者直接右键打开Git Bash，此时一定有个问题，假如用户不是用户管理员账户会报错，提示无法在此处创建文件夹。建议直接使用Git Bash

  输入一下命令：

1. npm install hexo-cli -g

2. npm install hexo –save

  依次执行完上述命令后，检查hexo是否安装完毕。输入：hexo -v显示当前安装hexo的版本号等信息说明安装完毕


## 创建Hexo工程 ##

1. 在适合位置创建个文件夹用于初始化Hexo工程(强烈建议使用英文而非中文)，例如：D:\hexoProject。依照Hexo的安装打开Git Bash。

2. 输入命令：hexo init;

3. 输入命令：hexo generate

4. 输入命令：hexo server

此时本地版的Hexo已搭建并启动可以在浏览器中输入:http://localhost:4000进行访问。



## Github的配置 ##

  这个时候您刚才申请的github账号需要用到了，下面我将带您一步步创建仓库，并能进行简单的访问。

  在github上点击右侧的New repository创建仓库，仓库名为yourName.github.io(注意：这里的yourName与Owner相同)，其他为默认配置。

&emsp 进入仓库后您会发现此时仓库内什么内容也没有，您需要在上方的功能栏上点击Setting，在Setting中的Github Pages下点击Choose a theme，进入后随便选择个主题即可，这个在稍后会覆盖。再次返回仓库后您会发现仓库中有了文件，到此您的github仓库已经完全配置好了，稍等几分钟您就可以通过yourName.github.io进行访问了。



## Hexo上传到GitHub ##

1. 在您刚才创建的Hexo工程的文件夹下找到_config.yml。
2. 修改_config.yml，在文件的末位您会发现deploy: type:，您需要对此进行配置 完整配置为一下内容(注意冒号的后面必须要有空格)

deploy:

  type:git

  repository:git@github,com:yourName/yourName.github.io

  branch:master

3. 到此所有的配置已搞定，当然您也可以在当前文件夹下修改title等配置进行一个完整的配置

4. 在Hexo工程下打开Git Bash,进行此次文件上传的github账户配置

  (1)输入命令 git config –global user.name “yourName”

  (2)输入命令 git config –global user.email “yourEmail”(此处的yourEmail为您注册github的Email)

  (3)输入命令 npm install hexo-deployer-git –save

  (4)输入命令 hexo clean

  (5)输入命令 hexo g

  (6)输入命令 hexo deploy

5. 此时再去打开yourName.github.io就可以发现本地的blog已经在网上显示了


## hexo一些常用命令 ##：

  hexo new “postName” #新建文章

  hexo new page “pageName” #新建页面

  hexo generate #生成静态页面至public目录

  hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

  hexo deploy #将.deploy目录部署到GitHub

  hexo help #查看帮助

  hexo version #查看Hexo的版本


  至此hexo已搭建完成本地博客并上传github并外网显示已全部完成。下节将讲述些Hexo的配置及githubPage的域名绑定。
